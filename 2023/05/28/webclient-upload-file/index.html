Spring-cloud-gateway-mvc 解决文件上传

背景

我们有个管理后台的服务使用 spring-cloud-gateway-mvc 作为网关代理到其他服务来管理系统。我们有个上传文件的场景，但是由于 spring-cloud-gateway-mvc 并没有把上传的文件代理到其他服务。解决的办法有两种方式：

  • 通过生成对象存储的预签名的方式
  • 用webClient上传文件
