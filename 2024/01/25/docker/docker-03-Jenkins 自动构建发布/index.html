Jenkins CI/CD

使用 Jenkins 的 pipeline 来实现一个 SpringBoot 项目自动打包与部署。大致思路是，先用 Maven 打成 Jar 包，然后使用 Dockerfile 来构建镜像并上传到自己的 Docker Hub 仓库中，然后再拉去镜像并启动容器。

1. 安装 Jenkins

首先要增加 Jenkins 的仓库源

访问本机8080端口

2. 创建 Pipeline 的任务

首先通过 git 拉取代码，这里需要到凭证管理去添加(创建 SSH 的私钥和公钥)；然后通过 Maven 打包；然后通过 Dockerfile 构建镜像，并上传镜像到自己的 Docker Hub 仓库，这里需要登陆 Docker Hub ，所以配置密码变量，如何配置可以参考：**Hiding password in Jenkins pipeline script ；**最后就去拉取镜像，启动容器。
